# ClawdRave

Part of **[ClawdRave](../README.md)** — a safe and efficient framework for collaborative vibecoding. This folder is the **Mission Control** UI: dual agents (Kai / Nova), bridge, security feed, and demo or live modes.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Dev server on `0.0.0.0:3000` |
| `npm run bridge` | Run `server/bridge.ts` (WebSocket bridge) |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |

Persona and prompt files: **`agents/kai/`**, **`agents/nova/`**. For Lightning + OpenClaw hosting, see the [root README](../README.md).
